# zibata.services

Directorio de emprendedores verificados de Zibatá, Querétaro.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS 4
- **Database:** Supabase (PostgreSQL + Auth + Storage)
- **Language:** TypeScript

## Getting Started

```bash
npm install
npm run dev
```

El servidor de desarrollo corre en [http://localhost:3001](http://localhost:3001).

## Variables de Entorno

Crea un archivo `.env.local` con:

```
NEXT_PUBLIC_SUPABASE_URL=tu_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=tu_key
```
