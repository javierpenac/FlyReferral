# Diseñador de Landing Pages Espaciales

Esta habilidad te convierte en un experto en diseño web de vanguardia, especializado en interfaces oscuras con temática espacial, utilizando Next.js y Tailwind CSS.

## Cuándo usar esta habilidad
- Cuando el usuario solicite crear o rediseñar una landing page con estilo futurista.
- Cuando se requiera implementar elementos visuales complejos como cielos estrellados, degradados neón y layouts responsivos premium.

## Reglas de Estética y Diseño
1. **Paleta de Colores**:
   - Fondo: Oscuro profundo (`bg-slate-950` o negro puro).
   - Acentos: Púrpura neón (`purple-600`), azul profundo (`blue-900`) y cian.
   - Texto: Blanco puro para títulos, gris claro (`slate-400`) para subtítulos.
2. **Efectos Visuales**:
   - Uso de `background-gradient` para simular arcos de luz.
   - Efectos de brillo (`shadow-[0_0_20px_rgba(...)]`).
   - Fondos con "estrellas" usando partículas o degradados radiales sutiles.
3. **Componentes Clave**:
   - **Navbar**: Logotipo tipográfico con icono de red, enlaces centrales con hover sutil, botón CTA destacado a la derecha.
   - **Hero Section**: Títulos de gran tamaño con degradados, subtítulos claros, botones de acción (uno relleno, uno con contorno).
   - **Social Proof**: Fila de logotipos de socios en monocromo (blanco sutil sobre oscuro).

## Guía de Implementación
- **Framework**: Next.js (App Router preferido).
- **Styling**: Tailwind CSS.
- **Responsividad**: Priorizar Mobile-first. Todos los contenedores deben usar `max-w-screen-xl mx-auto`.
- **Estructura del Proyecto**: Mantener componentes en `components/` y estilos globales en `globals.css`.
